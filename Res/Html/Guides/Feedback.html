GitUp is a small tool that helps you stay in-sync with the projects you develop.

It is aimed to be tiny tool that is available anytime and anywhere.

GitUp it introduces features that are not generally supported by standart git cli like 'Daily StandUp filter' that groups commits by Authors and by Branches for easier analyzation of latest code changes.

If you like what GitUp does for you support its development by mentioning #GitUp in a tweet or rating app in the AppStore.

Thank you for using GitUp.
