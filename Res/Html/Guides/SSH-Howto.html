Connecting to private repos via SSH

GitUp allows you to connect to a server via ssh:// protocol and use its benefits like secured connection and user authorization without username and password sharing.

In order to connect to git server you'll need to provide ssh keys that you've generated to GitUp application.

(This guide assumes that your keys are generated and installed on server already. See how to do this for:

  • Github
  • Bitbucket
  • Custom server (privately hosted)

Follow the step below to install your ssh keys into GitUp:

  • Archive your id_rsa and id_rsa.pub files into ZIP archive
  • Name archive after server name the keys were generated for (e.g. Github.zip)
  • Connect your iPhone to computer and open iTunes
  • In iTunes navigate to iPhone ➟ Apps ➟ File Sharing
  • Select GitUp app from the list below and drag ZIP archive into GitUp Documents folder

That's it. ZIP archive has been copied into application.

Next time you fire GitUp it will detect new keys archive and ask you for the passphrase (password that was set during keys generation).

Make sure you set correct password. GitUp stores it securely within local keychain so no one except GitUp can read it.

ssh:// protocol will appear right next to other protocols list.

Happy SSH-ing! :)
